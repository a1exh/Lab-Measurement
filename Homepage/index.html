Lab::VISA

The Perl packages Lab::VISA, Lab::Instrument, and Lab::Tools offer instrument and measurement control from Perl via National Instrument's VISA library. Encapsulating the complexity of VISA calls into a straightforward to use library, they are designed to make any lab power user happy.

How to obtain

The packages are free software and can be downloaded from CPAN. Follow these links for

  • Lab::VISA,
  • Lab::Instrument, and
  • Lab::Tools.

Documentation

Quite some documentation is available online in HTML (here) and PDF (here) format. This documentation starts with a general introduction into using Lab::VISA and continues with a description of the various APIs.

Additionally, a powerpoint presentation is contained in the Lab::VISA package at Tutorial/Talk/Lab-VISA.ppt (TODO: make available here). The examples shown therein are located in the 'Tutorial' directory of that package.

There is a mailing list set up for Lab::VISA (lab-visa-users mailing list). This mailing list is also the right place to give feedback and ask for help.

Status

Although this software has been used for years in real measurements by its developers, it remains work in progress. Please bear with us while we constantly improve code and documentation.

Lab::VISA is currently developed at nanophysics group, Prof. J. P. Kotthaus, LMU München and mesoscopic physics group, Prof. C. Strunk, Uni Regensburg. User feedback indicates further uses in industrial and academic environments.

Authors

Lab::VISA and friends were originally developed by Daniel Schröer and are currently continued by Andreas K. Hüttel and Daniel Schröer.
